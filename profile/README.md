## 🚀 Getting Started

### Prerequisites
- Node.js (v16+)
- npm or yarn
- Git

### Quick Setup
```bash
# Clone a repository
git clone https://github.com/DevSnapz/[repo-name].git

# Navigate to project
cd [repo-name]

# Install dependencies
npm install

# Start development server
npm run dev
